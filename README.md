# 🚀 TenderTrace

A full-stack web application to bring **transparency and efficiency** to government tender management.

This system provides two separate dashboards:
- 🏛️ Government Dashboard
- 👷 User/Contractor Dashboard

Both are powered by a **shared backend API**.

---

## 📁 Project Structure (Monorepo)


---

## ✨ Features

### 🏛️ Government Side
- Create & manage tenders
- Update project progress
- Add bills and completion details
- Track contractor activities

### 👷 User Side
- View available tenders
- Apply for tenders
- Track application status
- Dashboard for personal activity

### ⚙️ Backend
- REST API using Express.js
- MongoDB database integration
- Centralized data handling for both apps
